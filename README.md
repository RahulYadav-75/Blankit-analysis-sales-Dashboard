# Blinkit Sales Dashboard (MS Excel)
# Overview
This project is an interactive **MS Excel dashboard** built to analyze Blinkit sales, outlet performance, product categories, and customer ratings.
## Dashboard Preview

![Dashboard](99ebc70b-e35d-468f-8d47-c2f33da7a34b.png)

## Key KPIs

-   **Total Sales:** \$1.20M
-   **Average Sales:** \$141
-   **Number of Items:** 8,523
-   **Average Rating:** 4.0

## Features

-   Interactive filters:
    -   Outlet Size
    -   Outlet Location
    -   Item Type
-   Sales trend by outlet establishment year
-   Fat content analysis
-   Item type sales comparison
-   Outlet size distribution
-   Outlet location comparison
-   Outlet type analysis (Sales, Average Sales, Number of Items)

## Dashboard Sections

### 1. KPI Cards

Quick summary of business performance.

### 2. Sales Trend

Shows yearly sales based on outlet establishment year.

### 3. Fat Content Analysis

Compares Low Fat and Regular products.

### 4. Item Type Analysis

Displays sales by product category.

### 5. Outlet Analysis

Analyzes outlet size, location, and outlet type.

## Tools Used

-   Power BI
-   Power Query
-   DAX
-   Microsoft Excel / CSV

## Business Insights

-   Medium outlets generate the highest sales.
-   Tier 3 locations contribute the largest sales.
-   Fruits & Vegetables and Snack Foods are top-selling categories.
-   Low Fat products account for a larger share of sales.
-   Average customer rating is 4.0.
