# Blinkit Sales Dashboard (MS Excel)
# 📌Project Overview
This project is an interactive **MS Excel dashboard** built to analyze Blinkit sales, outlet performance, product categories, and customer ratings.
## Dashboard Preview

📸![Dashboard](https://github.com/RahulYadav-75/Blankit-analysis-sales-Dashboard/blob/main/Blinkit%20dashboard.png)

## 📈Key KPIs

-   **Total Sales:** \$1.20M
-   **Average Sales:** \$141
-   **Number of Items:** 8,523
-   **Average Rating:** 4.0

## Features

-   Interactive filters:
    -   Outlet Size
    -   Outlet Location
    -   Item Type
-   Sales trend by outlet establishment year
-   Fat content analysis
-   Item type sales comparison
-   Outlet size distribution
-   Outlet location comparison
-   Outlet type analysis (Sales, Average Sales, Number of Items)

## Dashboard Sections

### 1. KPI Cards

Quick summary of business performance.

### 2. Sales Trend

Shows yearly sales based on outlet establishment year.

### 3. Fat Content Analysis

Compares Low Fat and Regular products.

### 4. Item Type Analysis

Displays sales by product category.

### 5. Outlet Analysis

Analyzes outlet size, location, and outlet type.

## 💡Business Insights

-   Medium outlets generate the highest sales.
-   Tier 3 locations contribute the largest sales.
-   Fruits & Vegetables and Snack Foods are top-selling categories.
-   Low Fat products account for a larger share of sales.
-   Average customer rating is 4.0.
  

## 📂Project Flow

### 1. Data Collection

* Downloaded the **Blinkit Grocery Sales** dataset from **Kaggle**.
* Imported the dataset into Microsoft Excel for initial analysis.

### 2. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data entries.
* Standardized column names and data formats.
* Verified data accuracy before analysis.

### 3. Data Preprocessing

* Transformed and cleaned the data using **Power Query**.
* Created calculated columns and measures using **DAX**.
* Built data relationships where required.
* Optimized the dataset for reporting and visualization.

### 4. Data Visualization

Developed an interactive **Power BI Dashboard** with the following features:

#### 📈KPI Cards

* Total Sales
* Average Sales
* Number of Items
* Average Rating

#### 📉Visualizations

* Sales Trend by Outlet Establishment Year
* Fat Content Analysis
* Item Type Analysis
* Outlet Size Distribution
* Outlet Location Analysis
* Outlet Type Performance

#### Interactive Filters (Slicers)

* Outlet Size
* Outlet Location
* Item Type
